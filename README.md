# 💪 Grip Forge

一个纯前端握力训练计时器。零依赖，开箱即用。

## 🚀 在线体验

👉 [https://leonsux.github.io/grip-forge/](https://leonsux.github.io/grip-forge/)

## ✨ 功能

- 🔥 圆环倒计时动画（锻炼红色 / 休息绿色）
- 🎯 每组独立：做完一组自动停止，手动开始下一组
- 📊 日历热力图，直观查看训练频率
- 🔥 连续打卡天数统计
- ⚙️ 自定义组数、次数、锻炼时间、休息时间
- 💾 数据存储在 localStorage，无需后端
- 🔊 倒计时提示音 + 完成音效 + 撒花动画
- ⌨️ 键盘快捷键：空格开始/暂停，R 重置
- 📤 支持导出训练数据为 JSON
- 📱 移动端适配
